Work with Github

  • git init - initialization empty repository
  • git add (filename) - adding file in remote repository
  • git branch (branch name) - create branch
  • git checkout - change branch
  • git reset (--soft --medium --hard) - delete commit (commits)
  • git commit -m "message" - create new commit
  • git push origin (your branch) - push in remote repository
  • git remote add origin (SSH link or HTTPS link) - connection with repository on Github
  • git log - watch commit history
  • git diff - Show changes working tree and the index
